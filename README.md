# iPhone Resale Website

A modern iPhone resale website built using HTML, CSS, and JavaScript. This project displays available iPhones, pricing, and allows users to add products to a shopping cart.

## Features

- Clean and simple Apple-style design
- Product listings for iPhones
- Add to Cart functionality
- Responsive layout
- Lightweight and fast website

## Technologies Used

- HTML
- CSS
- JavaScript

## Purpose

This project demonstrates a simple e-commerce style website for reselling smartphones online.

## Future Improvements

- Online payment integration
- User accounts
- Inventory management
- Product image gallery
- Mobile optimization
